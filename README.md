# python-script-ssh-auto
python-script-ssh-auto

script support for python 2

install

1. open command line 
2. pip install paramiko
3. edit file name ssh_auto.py change  
SSH_ADDRESS = "HOST_IP",   
SSH_USERNAME = "USERNAME",   
SSH_PASSWORD = "PASSWORD"   
  
4.save file 
5.run script in command (python ssh_auto.py)
